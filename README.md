# 🛡 Anamo-Lie

AI-powered login anomaly detection system for cybersecurity threat prevention.

## Features

- Real-time anomaly detection
- Risk scoring
- Behavioral analysis
- Threat visualization dashboard
- Brute force attack detection

## Tech Stack

- Streamlit
- Python
- Scikit-learn
- Plotly

## AI Model

Isolation Forest anomaly detection algorithm.
